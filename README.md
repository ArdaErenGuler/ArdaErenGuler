# Merhaba, Ben Arda! 👋

Gazi Üniversitesi'nde Yönetim Bilişim Sistemleri (MIS) öğrencisiyim. Yazılım süreçlerini ve veri dünyasını harmanlamayı seviyor, bu alanlarda kendimi geliştiriyorum.

### 🚀 Neler Yapıyorum?
* 🌱 Web tabanlı otomasyonlar ve veritabanı yönetimi üzerine çalışıyorum.
* 💡 **DevOps** süreçlerine ve kültürüne büyük ilgi duyuyor, yazılım geliştirme ile operasyon adımlarını optimize etme konularında fikirler üretiyorum.
* 📊 **Veri Analizi** ile ilgileniyor; Excel Dashboard ve Pivot Table kullanarak veriden değer yaratma süreçlerini takip ediyorum.

### 💻 Teknolojiler ve Araçlar
**Diller & Backend (Temel-Orta Seviye):** PHP, Python, C#, SQL  
**Frontend:** HTML, CSS, Bootstrap, JavaScript  
**Veri Analizi:** MS Excel (Pivot Table, Dashboard, Veri Görselleştirme)
